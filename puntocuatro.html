Glosario de etiquetas HTML

  1. ol: Se utiliza para crear una lista ordenada de elementos en una página web. Los elementos de la lista se numeran automáticamente. Para crear un glosario de términos con la etiqueta "< ol >", se pueden usar los elementos "< li >" para definir cada término y su definición.

  2. br: Se utiliza para insertar un salto de línea en una página web. La etiqueta "< br >" no tiene etiqueta de cierre porque es una etiqueta vacía. Se puede usar para separar visualmente los elementos de una lista, una dirección o cualquier otro bloque de texto. Para crear un glosario de términos con la etiqueta ‘< br> ’, se puede usar esta etiqueta para separar visualmente cada término y su definición en líneas diferentes.

  3. img: Se utiliza para insertar una imagen en una página web. Se usa para especificar la ubicación de la imagen, su tamaño y su descripción. Para crear una imagen con la etiqueta , se necesita especificar la ruta de la imagen en el atributo src. El atributo alt se utiliza para proporcionar una descripción de la imagen que se muestra si la imagen no se puede cargar. También se pueden especificar otros atributos, como el tamaño de la imagen, el borde y el espaciado.

  4. header: Se utiliza para definir la cabecera o encabezado de una sección o página web. El contenido que se encuentra dentro de esta etiqueta puede incluir un logotipo, un título, una barra de navegación y otros elementos que se mostrarán en la parte superior de la página. La etiqueta < header> es un elemento de bloque y se utiliza típicamente en conjunto con otras etiquetas como < nav> y < h1> para crear una cabecera estructurada y semántica.

  5. b: Se utiliza para aplicar negrita al texto en una página web. Esta etiqueta no debe utilizarse para crear un título o encabezado, sino más bien para resaltar texto dentro de un párrafo. La etiqueta < b> se puede utilizar en combinación con otras etiquetas como < p> y < span> para aplicar negrita a un bloque de texto o a una sección específica. Sin embargo, es importante recordar que el uso excesivo de la etiqueta < b> puede hacer que el contenido sea difícil de leer y comprender.

  6. form: Se utiliza para crear un formulario en una página web. Un formulario permite al usuario ingresar datos y enviarlos al servidor para ser procesados. La etiqueta < form> establece el inicio y el final del formulario y puede incluir varios elementos, como campos de texto, botones de opción, botones de radio y menús desplegables. La etiqueta < form> también puede incluir atributos como action y method para especificar la acción y el método de envío del formulario. El contenido del formulario se procesa generalmente en el lado del servidor mediante lenguajes de programación como PHP y Python.

  7. table: Se utiliza para crear una tabla en una página web. Una tabla se compone de filas y columnas y se utiliza a menudo para presentar información en una forma organizada y estructurada. La etiqueta < table> establece el inicio y el final de la tabla y puede incluir varios elementos, como las etiquetas < tr> para filas, < th> para encabezados de columna y < td> para celdas de datos. La etiqueta < table> también puede incluir atributos para especificar el ancho de la tabla, el borde y otros estilos visuales. Es importante utilizar la etiqueta < table> de forma semántica y estructurada para mejorar la accesibilidad y la comprensión del contenido para los usuarios y los motores de búsqueda.

  8. input: Se utiliza para crear campos interactivos en una página web, como cajas de texto, botones de opción, casillas de verificación y menús desplegables. La etiqueta < input> es una etiqueta vacía que se utiliza con varios tipos de valores de atributo para definir el tipo de campo de entrada que se va a crear. Algunos de los tipos más comunes incluyen text, password, radio, checkbox, submit y select. La etiqueta < input> también puede incluir otros atributos como name para identificar el campo, value para proporcionar un valor predeterminado y required para hacer que el campo sea obligatorio. Los datos ingresados en los campos de entrada se pueden enviar al servidor para su procesamiento mediante un formulario.

  9. button: Se utiliza para crear un botón interactivo en una página web que puede realizar una acción cuando se hace clic en él. La etiqueta < button> establece el inicio y el final del botón y puede incluir texto, imágenes u otro contenido entre las etiquetas. La etiqueta < button> también puede incluir atributos como type para especificar el tipo de botón, como submit, reset o button. Los botones se utilizan a menudo para enviar formularios al servidor, realizar una acción en la página o cambiar el estado de la página en función de la interacción del usuario. La etiqueta < button> también se puede combinar con JavaScript para crear interacciones más complejas y personalizadas en una página web.

  10. label: Se utiliza para asociar una etiqueta de texto con un elemento de formulario, como una casilla de verificación, un botón de opción o una caja de texto. La etiqueta < label> establece el inicio y el final del texto de la etiqueta y se vincula con el elemento de formulario utilizando el atributo for o mediante la inclusión del elemento de formulario dentro de la etiqueta < label>. Cuando el usuario hace clic en la etiqueta de texto, el elemento de formulario asociado recibe el enfoque. La etiqueta < label> también se puede utilizar para proporcionar información adicional sobre el elemento de formulario, como instrucciones de entrada o explicaciones adicionales.
